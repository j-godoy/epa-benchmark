command to run: 

Rscript "boxplot_pit.R" "path/to/results.csv" "ALL"


notes:
 - ALL: generates boxplots for all subjects in csv file. An alternative is put desired subjects as parameters. Example: "com.example.socket.Socket" "com.example.listitr.ListItr"
 - Rscript: In Windows, use path/to/Rscript.exe
